# Predicting-Liquor-Sales-Trends-in-Iowa
This project focuses on predicting monthly liquor sales for retailers in Iowa using historical transaction data from the Iowa Alcoholic Beverages Division. The objective was to build a forecasting pipeline that could help businesses better understand sales behavior, capture seasonal demand patterns, and support smarter inventory and supply chain decisions at the store level.

The analysis began with extensive data cleaning, aggregation, and feature engineering in Python. Raw transaction-level data was transformed into store-level monthly sales data, with additional variables created to improve predictive performance, including lag-based features, seasonal indicators, and geographic information. Exploratory analysis was also used to examine sales distributions, category-level performance, store-level trends, and recurring seasonal peaks, especially around year-end demand.

To forecast future sales, multiple machine learning models were tested, including Lasso, Ridge, Random Forest, SVR, and XGBoost. The final workflow used Bayesian hyperparameter optimization and model evaluation techniques such as RMSE, MAE, R², bootstrapping, and paired t-tests to compare performance. The results showed that Bayesian-tuned XGBoost delivered the strongest predictive accuracy, demonstrating the value of non-linear modeling for capturing seasonality and store-level variation in retail sales.
